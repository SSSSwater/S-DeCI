# 多中心fMRI因果-双曲原型辅助诊断系统：功能实现与代码衔接指南

## 一、 基础环境与核心依赖库搭建

本项目涉及欧氏特征提取、图神经网络以及非欧几何流形上的张量运算。为了让不同模块的代码能够在一个框架下顺畅运行，请确保使用统一的底层张量库。

* **深度学习底层：** `PyTorch` (建议 2.0 以上版本，支持高级自动微分)。
* **双曲几何基础运算库：** 必须安装 `Geoopt`。这是目前官方最权威的PyTorch黎曼流形优化与运算扩展库，HPEC原论文中庞加莱球（Poincaré Ball）的指数映射、对数映射等底层代码均基于此库构建。
* *安装命令：* `pip install geoopt`



## 二、 核心模块实现步骤与源码衔接方案

### 模块 1：DeCI前端时序信号解耦 (Temporal Decoupling)

* **功能目标：** 将输入的 $N \times T$ 原始BOLD fMRI信号，提取为剥离了设备漂移噪声的 $N \times 64$ 维纯净高频神经表征 $C$。
* **参考开源库：** `https://github.com/Levi-Ackman/DeCI`。
* **代码提取与衔接步骤：**
1. 从DeCI源码中提取 `Cycle and Drift Decomposition` 以及 `Channel-Independence` 模块。
2. **修改输出层：** 原DeCI代码的末端包含全连接分类头（Classification Head），直接输出分类Logits。您需要将网络**在Pooling或最后一层投影处截断**，确保它仅输出维度为 $(Batch, N, 64)$ 的特征张量 $C$，并将代表设备噪声的 Drift 特征丢弃。



### 模块 2：端到端可微潜在因果图学习 (Differentiable DAG Learning)

* **功能目标：** 在64维隐空间中，利用特征 $C$ 重构脑区间的因果邻接矩阵 $A \in \mathbb{R}^{N \times N}$，并在计算图中引入无环（DAG）惩罚。
* **当前实现参考开源库：**
  - `https://github.com/xiangyu-sun-789/NTS-NOTEARS`
  - `https://github.com/sharpenb/Differentiable-DAG-Sampling`
* **方法切换：** 模块 2 支持通过 `causal_graph_method` 在两种结构间切换：
  - `nts_notears`：使用 NTS-NOTEARS 风格的第一层因果权重与局部重构。
  - `dag_sampling`：使用 Differentiable-DAG-Sampling 风格的 edge logits、可微 permutation/Sinkhorn 拓扑顺序 mask 与 masked decoder。
* **代码提取与衔接步骤：**
1. **方法一：NTS-NOTEARS 第一层因果权重。** 不直接学习一个裸邻接矩阵 `self.A`。参考 NTS-NOTEARS，在 PyTorch 的 `nn.Module` 中定义两组非负第一层权重 `fc1_pos` 与 `fc1_neg`，用 `fc1_pos - fc1_neg` 表示 signed causal effect。对角线使用 mask 置 0，避免自环。
2. **NTS-NOTEARS 邻接矩阵提取：** 按 NTS-NOTEARS 的 `fc1_to_adj` 逻辑，从第一层权重中提取邻接矩阵。具体做法是对 hidden 维度求平方和并转置，得到 $A[parent, child]$：
   `A = sqrt(sum((fc1_pos - fc1_neg)^2, hidden_dim))`。
   该矩阵表示全局共享的脑区因果拓扑，形状为 $(N, N)$。
3. **方法二：Differentiable-DAG-Sampling 拓扑顺序 mask。** 参考 `ProbabilisticDAG`，定义可学习的 `perm_weights` 和 `edge_log_params`。`perm_weights` 通过 Sinkhorn 归一化得到近似 permutation 矩阵 $P$，再用上三角 mask 构造可微拓扑顺序约束：
   `order_mask = P^T @ upper_triangular_mask @ P`。
   `edge_log_params` 经 `sigmoid` 得到边概率，最终邻接矩阵为：
   `A = sigmoid(edge_log_params) * order_mask`。
4. **Differentiable-DAG-Sampling 重构特征：** 参考 `MaskedAutoencoder`，对每个 child node 使用 `A^T` 形成 masked input，再通过 node-wise local decoder 重构 $C_{hat}$。该方法更强调“先学习拓扑顺序，再在允许的拓扑方向上学习边概率”。
5. **DAG 损失函数：** 两种方法都保留可选 DAG penalty，默认可使用 NOTEARS 形式的无环惩罚：
   `L_DAG = trace(exp(A ∘ A)) - N`。
   对于 116 个脑区这种较大规模，也可切换到 `docs/新DAG因果方法.md` 中的 analytic DAG 约束，以缓解高阶环路梯度衰减和数值不稳定。对于 `dag_sampling`，拓扑顺序 mask 本身已经鼓励 DAG 结构，DAG penalty 可作为额外诊断或软约束。
6. **稀疏正则：** `nts_notears` 的 L1 sparsity 参考 NTS-NOTEARS 的 `fc1_l1_reg`，作用于第一层 `fc1_pos + fc1_neg`；`dag_sampling` 的 L1 sparsity 作用于最终 masked adjacency，鼓励边概率稀疏。
7. **对比实验：** 两种方法应保持相同输入 $C$、相同 reconstruction/DAG/L1 loss 权重和相同可视化输出，方便直接比较 `A_learned`、`A_learned_binary`、`C_hat`、重构误差和分类结果。
8. **衔接点：** 将 $C$、$C_{hat}$ 和学到的 $A$ 缓存在模块 2 中；当前阶段只学习因果关系，分类仍由模块 1 的 Cycle/seasonal 分支完成。模块 3 开始后再将 $A$ 传递给后续图卷积模块。



### 模块 3：双曲映射、莫比乌斯图卷积与Fréchet均值读取 (Hyperbolic GCN & Readout)

* **功能目标：** 对因果特征执行Backclip限幅，投影至庞加莱球，利用因果图 $A$ 进行双曲信息传递，最终压缩出全脑拓扑重心 $z_{global}$。
* **参考开源库：**
* 可微Fréchet均值：`https://github.com/CUVL/Differentiable-Frechet-Mean` (ICML 2020)。


* **代码提取与衔接步骤：**
1. **编写Backclip层：** 参考HPEC原著逻辑，编写一个自定义函数。获取每个脑区向量的欧氏范数，计算动态收缩因子 $\zeta_{c_i}$，并执行截断：`c_clipped = torch.max(1.0, r / norm) * c`。
2. **双曲投影：** 使用Geoopt库的接口：`manifold = geoopt.PoincareBall(c=1.0)`，接着执行 `h_0 = manifold.expmap0(c_clipped)`，将数据送入流形。
3. **双曲图卷积 (HGCN)：** 核心是莫比乌斯矩阵乘法。调用 `manifold.mobius_matvec(W, h_0)` 将特征升维至128维；再结合上一步学到的因果矩阵 $A$，使用 `manifold.mobius_add` 进行邻居特征聚合。
4. **图读取衔接：** 引入可微的Fréchet均值求解器。将聚合后的 $116$ 个节点的张量传入求解器，它会通过内部的黎曼优化迭代，返回唯一的128维双曲中心点 $z_{global} \in \mathbb{B}^{128}$。



### 模块 4：HPEC蕴含锥对齐与联合损失分类 (HPEC Classification & Loss)

* **功能目标：** 利用纯角度计算代替传统的预测，构建疾病亚型原型，并反向传播更新整个系统。
* **参考开源库：** `https://github.com/samuelefonio/HPEC` (AISTATS 2025，HPEC官方源码)。
* **代码提取与衔接步骤：**
1. **定义原型与孔径：** 从HPEC库中提取原型初始化代码（最大化余弦分离初始化），设定原型半径 $r^*$ 和孔径函数 $\psi$ 的计算模块。
2. **角度能量惩罚计算：** 移植共形角度计算函数 $\Xi(\pi_k, z_{global})$。这部分代码只需在欧氏切空间使用传统的内积即可无损计算，非常高效。根据角度和孔径实现能量损失 $E = \max(0, \Xi - \psi)$。
3. **预测输出：** 在推理（Inference）阶段，执行 `argmin(E)`，输出能量最小的原型类别。
4. **构建联合损失：** 在总 `loss.py` 中，将上述所有模块的损失相加：
`Loss_total = Loss_HPEC(z_global, label) + alpha * Loss_Recon(C, C_hat) + lambda * Loss_DAG(self.A) + gamma * L1_Norm(self.A)`
5. **联合优化器：** 由于普通参数（如DeCI权重）在欧氏空间，而双曲特征或双曲原型可能需要黎曼优化，建议使用 `geoopt.optim.RiemannianAdam` 作为全局优化器，它可以兼容处理欧氏与双曲参数。



## 三、 数据流与张量形状 (Tensor Shape) 追踪复核

为了防止代码组合时出现张量不匹配的Bug，请在每个阶段打印（`print(tensor.shape)`）并核对以下关键形状（假设Batch Size为 $B$，感兴趣脑区 $N=116$）：

1. `X_input`: $(B, 116, T)$ -> 原始fMRI信号
2. `C_out` (来自DeCI): $(B, 116, 64)$ -> 纯净高频信号
3. `A_matrix` (因果参数): $(116, 116)$ -> 全局共享的可学习因果拓扑
4. `C_hat` (重构特征): $(B, 116, 64)$
5. `H_proj` (进入庞加莱球): $(B, 116, 64)$ -> 属于Geoopt Manifold Tensor
6. `H_gcn` (升维聚合后): $(B, 116, 128)$
7. `Z_global` (Fréchet均值后): $(B, 128)$ -> 每个病人的全脑单一代表点
8. `Energy_logits`: $(B, K)$ -> $K$ 个疾病亚型的能量惩罚值，用于最终决策。

## 模块 2 现行正式路径补充：Temporal NTS-NOTEARS

当前模块 2 的正式实现已经从静态特征重构式图学习改为 **Temporal NTS-NOTEARS**。核心点如下：

- 模块 1 输出时间序列，模块 2 直接使用历史窗口 `x_{t-1}, ..., x_{t-L}` 预测 `x_t`。
- 主图为 `A_lag: [L, N, N]`，表示过去脑区到未来脑区的方向关系。
- `A0: [N, N]` 仅表示同一时间片残余依赖，仍保留弱 DAGMA / NOTEARS 风格约束。
- `A_lag` 使用第一层正负权重分解得到，保持 NTS-NOTEARS 的可微图学习思想。
- `A_lag` 不强制无环，因为时间顺序已经给出了方向。
- 下游模块 3 / GCN fallback 默认使用 `A_lag.mean(dim=0)`，并可按比例与样本相关矩阵融合形成 `A_cls`。

模块 2 的正式损失为：

```text
L_module2 = L_pred + lambda_sparse * L1(A_lag, A0)
          + lambda_smooth * smooth(A_lag)
          + lambda_dag * DAG(A0)
```

`static_feature`、`dag_sampling` 和旧 reconstruction 路径仅保留为 legacy/debug，不再是默认正式路径。
