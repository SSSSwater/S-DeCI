# ALFF/fALFF 跨数据集差异比较

本次比较用于判断模块 1 的 ALFF/fALFF 特征是否比原始时序统计更能压缩不同数据集之间的非生理差异。

## 比较设置

- 数据集：`Abide`、`MDD`、`Mātai`、`Neurocon`、`PPMI`、`Taowu`
- 模板：`AAL116`
- 每个数据集平衡抽样：40 个样本
- 原始信号统计：每个 ROI 的均值、标准差、一阶差分幅度、一阶差分标准差、最小值、最大值
- ALFF/fALFF 特征：每个 ROI 的 `log(1 + ALFF)`、`fALFF`、低频振幅波动、主导低频、时序标准差
- 距离：数据集级特征分布的均值距离与协方差 Frobenius 距离

## 主要结果

| 特征空间 | 平均均值距离 | 平均协方差距离 |
| --- | ---: | ---: |
| 原始信号统计 | 9.950 | 53.252 |
| ALFF/fALFF | 4.985 | 3.472 |
| ALFF / 原始比例 | 0.501 | 0.065 |

## 结论

ALFF/fALFF 后的数据集间均值距离约为原始统计的 50.1%，协方差距离约为原始统计的 6.5%。这说明当前模块 1 的 ALFF/fALFF 特征显著压缩了不同数据集之间的整体分布差异，尤其是协方差结构差异。

从这个结果看，ALFF/fALFF 确实比直接使用原信号统计更像稳定的低频生理活动描述，能在一定程度上排除高频噪声、尺度差异和部分采集差异。不过 MDD 与其他数据集之间的 ALFF 均值距离仍较大，说明多站点/疾病组构成差异仍然存在，后续仍需要结合站点 harmonization 或模块 2/3 的结构约束继续处理。
