# LP-Brain-HPEC 架构开发规范与模块分解书

**目标描述：**
本项目旨在实现一个跨双曲流形（Lorentz to Poincaré）的有向图卷积与原型分类网络，用于 fMRI/EEG 脑功能网络的因果拓扑分析。请严格按照以下模块定义、张量维度（Shape）和数学约束生成基于 `PyTorch` 和 `geoopt` 的代码。

**全局工程约束：**

1. **数值精度**：所有涉及双曲流形计算（指数映射、对数映射、测地线距离）的张量，**必须**强制使用 `torch.float64`，以防止靠近流形边界时的数值溢出（NaN）。
2. **底层库**：强烈建议继承或封装 `geoopt.manifolds.Lorentz` 和 `geoopt.manifolds.PoincareBall`。
3. **曲率设定**：定义全局可学习或固定的曲率参数 $K > 0$（代码中应保证曲率参数在正数域更新，如使用 `Softplus`）。

## 模块 1：数据预处理与特征输入接口 (Data I/O)

该模块负责衔接外部已算好的数据，不涉及复杂网络参数。

* **输入 1：频域节点特征** `X_euclidean`
* **Shape**: `[B, N, d_in]`（$B$: Batch size, $N$: 脑区节点数, $d_{in}$: 全频段 PSD 采样点数，如 32 或 64）。


* **输入 2：有向因果邻接矩阵** `A_causal`
* **说明**：由 NOTEARS 算法生成的非对称邻接矩阵。
* **Shape**: `[B, N, N]`。$A[i, j]$ 表示节点 $i$ 对节点 $j$ 的因果驱动力。


* **处理逻辑：洛伦兹提升 (Lorentz Lifting)**
* 将欧氏特征 $x \in \mathbb{R}^{d_{in}}$ 映射到洛伦兹流形 $\mathbb{H}_K^{d}$ 原点 $O = (\sqrt{1/K}, 0, \dots, 0)$ 的切空间，并通过指数映射投射到流形上。
* **计算公式**: $x^{0,H} = \exp_o^K((0, x^{0,E}W_{lift}))$，其中 $W_{lift}$ 为线性投影矩阵。
* **输出** `X_lorentz_0`: Shape `[B, N, d+1]`。

## 模块 2：有向洛伦兹符号图卷积层 (Directed Lorentz GCN)

因果矩阵是非对称的，要求 Agent 将图卷积拆分为“入度（In-degree）”和“出度（Out-degree）”两组独立的洛伦兹注意力机制。

* **输入**: 上一层特征 `X_lorentz_{l-1}` `[B, N, d+1]`，有向矩阵 `A_causal`。
* **处理逻辑 (要求 Agent 在 $T_O\mathbb{H}_K^d$ 切空间或局部切空间进行)**:
1. **切空间特征映射**:
提取当前节点 $i$ 的对数映射 $h_i = \log_O^K(x_i^{l-1, H})$。
2. **双向注意力计算 (Lorentzian Attention)**:
分别针对出边（源节点为 $i$）和入边（目标节点为 $i$）计算注意力权重。
* 权重公式：利用洛伦兹内积 $\langle \cdot, \cdot \rangle_{\mathcal{L}}$ 计算 Query 和 Key 的相似度。
* 对出边邻居 $\mathcal{N}_{out}$ 和入边邻居 $\mathcal{N}_{in}$ 分别做 `Softmax` 归一化。


3. **有向特征聚合 (Aggregation)**:
在节点 $i$ 的局部切空间 $T_{x_i}\mathbb{H}_K^d$ 中进行：

$$\Delta_i = \sum_{j \in \mathcal{N}_{in}} w_{j \rightarrow i} \log_{x_i}^K(x_j) + \alpha \sum_{k \in \mathcal{N}_{out}} w_{i \rightarrow k} \log_{x_i}^K(x_k)$$



（$\alpha$ 为可学习的平衡系数）。
4. **流形更新**:

$$x_i^{l, H} = \exp_{x_i}^K(\Delta_i)$$




* **输出** `X_lorentz_l`: Shape `[B, N, d+1]`。

## 模块 3：切空间近似图池化 (Tangent Space Readout)

用 $\mathcal{O}(N)$ 的切空间池化代替耗时的 Fréchet 均值计算。

* **输入**: 最后一层图特征 `X_lorentz_L` `[B, N, d+1]`。
* **处理逻辑**:
1. 将所有节点映射回原点 $O$ 的切空间：
$V_{tangent} = \log_O^K(X\_lorentz\_L)$ （此时切空间向量可视为普通的欧氏向量）。
2. 在欧氏维度上进行 Average Pooling：
$v_{mean} = \frac{1}{N} \sum_{i=1}^N V_{tangent}[i]$。
3. 再将池化后的均值向量投射回洛伦兹流形：

$$\mu_{\mathcal{G}} = \exp_O^K(v_{mean})$$




* **输出** `Graph_Embedding_Lorentz`: Shape `[B, d+1]`。

## 模块 4：跨流形保角投影 (Stereographic Bridge)

这是整个架构的连接枢纽，要求 Agent 严格按照公式编写，不可调用未知的第三方 API。

* **输入**: 洛伦兹图嵌入 `Graph_Embedding_Lorentz`，假设其张量结构为 $(x_0, x_1, \dots, x_d)$。
* **处理逻辑**:
实现洛伦兹模型到庞加莱球（Poincaré Ball）的保角映射公式：

$$z_{poi} = \frac{(x_1, x_2, \dots, x_d)}{x_0 + \sqrt{1/K}}$$


* **约束**: 输出的张量维度将从 `d+1` 降维至 `d`。
* **输出** `Graph_Embedding_Poincare`: Shape `[B, d]`。

## 模块 5：MAC-HBR 防御与原型能量分类器

摒弃传统的线性分类层，利用蕴含锥（Entailment Cone）进行角度匹配。

* **参数定义**:
* **原型矩阵** `Prototypes`: 定义在庞加莱球中的类别原型 $\Pi = \{\pi_1, \pi_2, \dots, \pi_C\}$，Shape `[C, d]`。这些原型在训练初期随机初始化，并约束在一个固定半径 $r^*$ 的超球面上。


* **步骤 5.1：MAC (Möbius Annulus Clipping)**
* **输入**: `Graph_Embedding_Poincare` (记为 $z$)。
* **逻辑**: 计算欧氏范数 $||z||_2$。设定最小安全半径 $r_{min} = r^* + \epsilon$ 和最大半径 $r_{max} = 1 - 10^{-4}$。
* 若 $||z||_2 < r_{min}$：$z_{stable} = z \cdot (r_{min} / ||z||_2)$。
* 若 $||z||_2 > r_{max}$：$z_{stable} = z \cdot (r_{max} / ||z||_2)$。


* **步骤 5.2：角度与能量计算 (Energy Loss)**
* 计算 $z_{stable}$ 与目标类别原型 $\pi_y$ 之间的庞加莱角度 $\Xi(\pi_y, z_{stable})$。
* **孔径函数**: $\psi(\pi) = \arcsin\left(c \cdot \frac{1 - ||\pi||^2}{||\pi||}\right)$（$c$ 为常数，确保数学域安全）。
* **能量**: $E(\pi, z) = \max(0, \Xi(\pi, z) - \psi(\pi))$。


* **步骤 5.3：HBR 软惩罚计算**
* $\mathcal{L}_{HBR} = \frac{1}{B} \sum_{b=1}^B \max(0, \text{arctanh}(||z_{stable}^{(b)}||) - R_{safe})^2$。


* **输出**: 预测类别（根据最小能量 $E$ 决定）与联合 Loss 计算模块。

## 模块 6：总体损失函数集成 (Loss Configuration)

* **损失公式**: $\mathcal{L}_{total} = \mathcal{L}_{cls} + \lambda \mathcal{L}_{HBR}$。
* 其中分类损失为对比蕴含损失：

$$\mathcal{L}_{cls} = \frac{1}{B} \sum_{b=1}^B \left[ E(\pi_{y_b}, z^{(b)}) + \sum_{j \neq y_b} \max(0, \xi - E(\pi_j, z^{(b)})) \right]$$


* **说明**：$\lambda$ 为 HBR 的权重（如 0.1），$\xi$ 为对比间隔 Margin（如 1.0）。
